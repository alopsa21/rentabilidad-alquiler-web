# F3b-02 — Tarjetas acumulativas

## Objetivo

Cada análisis debe crear automáticamente una tarjeta en un panel.

---

## Comportamiento

- Resultado = nueva tarjeta
- No pedir confirmación
- Mantener array de búsquedas en state React

---

## Tareas

- Crear componente Card
- Mantener lista de resultados
- Renderizar tarjetas verticalmente
- Cada tarjeta muestra:
  - estado (verde/amarillo/rojo)
  - ubicación
  - precio
  - alquiler estimado

---

## Criterio de aceptación

- Analizar varias URLs crea varias tarjetas
- No se pierde el input
